Project README
===============
CompanyOS — Internal Company Platform (Project & Bug-fix Guide)

Short description

CompanyOS is a full‑stack internal workforce platform (attendance, task management, integrated chat, notifications, dashboards) built with React + TypeScript (Vite) front-end and Node.js + Express + Prisma + PostgreSQL back-end with Socket.io for real-time features. This repository contains the project specification, implementation notes, and a focused set of targeted bug fixes and deployment guidance.

What's included

Full product specification (modules: Auth, Users, Attendance, Tasks, Chat, Notifications, Dashboards, Settings).

Detailed Prisma database schema and API route list.

Socket.io event design and cron jobs.

Design system tokens (color palette, typography, component rules).

Folder structure for frontend (/client) and backend (/server).

Seed data plan and demo accounts.

A set of 11 targeted bug fixes for CompanyOS (clock-in/out, chat issues, route ordering, socket transports, dev proxy, UI loading states, etc.).

Free deployment plan and platform recommendations (Vercel, Railway/Render, Neon, Cloudinary).

QA and functional checklist for release readiness.
